# できたね！ジグソーパズル

幼稚園年長さん向けの、3 x 5 の横長ジグソーパズルです。ブラウザで `index.html` を開くだけで遊べます。

## 画像

次の3枚を使います。

- `assets/train-driver-view.png`
- `assets/plane-cockpit-view.png`
- `assets/bus-driver-view.png`

同じファイル名で差し替えると、ゲーム側はそのまま使えます。3枚とも `3:2` の横長画像に合わせています。
